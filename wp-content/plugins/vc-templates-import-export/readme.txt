Export & Import WPBakery Page Builder Templates (Saved Templates/My Templates) in few clicks.

== Description ==
Export & Import templates for <strong>WPBakery Page Builder</strong> (Saved Templates/My Templates) in few clicks.


This plugin is addon for <a href="http://bit.ly/vc-plugin" target="_blank"><strong>WPBakery Page Builder</strong></a>.


== Features ==
<ul>
	<li>
		<strong>Import/Export templates</strong> in JSON format – take your custom templates.
	</li>
	<li><strong>Single Export</strong> – export only those templates you wish.</li>
	<li><strong>Multi Choose</strong> – export multi choose templates you wish
		<a href="http://bit.ly/temport" target="_blank"><strong>(Premium version)</strong></a>.
	</li>
	<li><strong>Export All</strong> – Allow export all templates with a click
		<a href="http://bit.ly/temport" target="_blank"><strong>(Premium version)</strong></a>.
	</li>
<li><strong>Multisite (Network)</strong> – export & import templates from one place.
		<a href="http://bit.ly/temport" target="_blank"><strong>(Premium version)</strong></a>.
	</li>
</ul>

== Installation ==
1. Upload the plugin files to the "/wp-content/plugins/vc-templates-import-export/" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. WPBakery Page Builder -> Templates Import & Export screen to use the plugin.
